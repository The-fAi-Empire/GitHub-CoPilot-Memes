## üé® Meme Formats

### Format 1: Classic Meme Template

**Best for:** Logic fails, naming disasters

```
[TOP TEXT]
What I asked Copilot for
[IMAGE OF CODE]
[BOTTOM TEXT]
What Copilot gave me
```

**Example Templates:**
- Drake pointing
- Distracted boyfriend
- Two buttons
- This is fine

### Format 2: Code Screenshot + Reaction

**Best for:** Security oops, breaking changes

```
[CODE SCREENSHOT]
@username's Copilot suggestion
[REACTION EMOJI/IMAGE]
"It tried to delete production"
```

### Format 3: Story Format

**Best for:** Complex fails with context

```
Panel 1: What I wanted
Panel 2: What I typed
Panel 3: What Copilot suggested
Panel 4: The disaster
```

### Format 4: Side-by-Side Comparison

**Best for:** All categories

```
Expected:          |  Copilot's Reality:
[Good code]        |  [Cursed code]
                   |  @username
```

## üé® Design Guidelines

### Brand Colors

```css
Primary Purple:  #6e5494
Danger Red:      #ff6b6b
Success Teal:    #4ecdc4
Dark Background: #1a1a2e
Light Text:      #ffffff
Dark Text:       #2d3436
```

### Typography

- **Headings:** Bold, 48-72px
- **Body:** 24-36px
- **Username:** Bold, 20-28px
- **Font Families:** Impact, Arial Black, or Helvetica Bold

### Required Elements

Every meme MUST include:

1. **@username** - Feature the contributor prominently
2. **The Fail** - The actual disaster
3. **Visual Appeal** - Eye-catching design
4. **Hashtags** - #CoPilotDaily #GitLost
5. **Readable Text** - High contrast, large fonts

### Layout Best Practices

- **Keep it simple** - Don't overcrowd
- **High contrast** - Text must be readable
- **White space** - Let content breathe
- **Consistent style** - Match existing memes
- **Mobile-friendly** - Test on small screens

## üõ†Ô∏è Step-by-Step Process

### Step 1: Understand the Fail

1. Read the entire issue
2. Note the key moment of disaster
3. Identify the humor
4. Check what makes it shareable

### Step 2: Choose Format

Pick based on:
- Type of fail
- Available screenshots
- Story complexity
- Meme potential

### Step 3: Create the Meme

#### Using Canva:

1. Start with 1200x1200px blank canvas
2. Add background (use brand colors)
3. Insert code screenshot or text
4. Add reaction image/emoji
5. Include @username prominently
6. Add hashtags at bottom
7. Ensure high contrast

#### Using Figma:

1. Create 1200x1200px frame
2. Use components for consistency
3. Add your content
4. Export as PNG (high quality)

#### Using Photoshop/GIMP:

1. New file 1200x1200px, 72 DPI
2. Layer your elements
3. Add text with effects
4. Flatten and export

### Step 4: Review Checklist

Before posting:

- [ ] @username is clearly visible
- [ ] Text is readable (test on phone)
- [ ] Humor is appropriate
- [ ] No offensive content
- [ ] High quality image
- [ ] Hashtags included
- [ ] File size <500KB
- [ ] Follows brand guidelines

### Step 5: Submit

1. Save as PNG or JPG
2. Optimize image (TinyPNG)
3. Comment on the issue with meme
4. Add to website's assets folder
5. Tag the contributor
